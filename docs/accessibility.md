# Accessibility Guide

FlowSync includes key WCAG-oriented improvements:

- Skip link for keyboard users (`Skip to main dashboard`)
- Keyboard access for map overlay toggles and SVG zone nodes (Enter/Space)
- Escape key focus clear behavior
- ARIA labels on primary controls
- Live regions for status/chat announcements
- Visible focus outlines for keyboard navigation

## Quick check

1. Press `Tab` from page load and use the skip link.
2. Focus map toggle chips and press `Enter`/`Space`.
3. Navigate to a zone node in the SVG and press `Enter`.
